Collision handler: handles collisions for the player character, manages interactions with enemies, NPCs, and obstacles, including respawning the player at the start position when necessary. also integrates with the player's backpack system and resets relevant  game elements upon respawn.

CollectableManager: This script manages the spawning and collection of collectibles 
in the game, including coins, speed boosters, and shields.  handles the initial spawning of these items at set intervals  along the player's path and updates the coin count when coins are  collected. The manager also ensures that new collectibles are  spawned ahead of the player as they progress through the level.


collectableBase: This is the base class for all collectibles in the game.  handles the trigger detection and calls the abstract  OnCollected method which derived classes must implement  to define their specific collection behavior  (e.g. increasing score, granting power-ups, etc.).  The collectible will be destroyed upon successful collection.

PlayerCharacter: This script defines the behavior of the player character in the game.  handles player movement, jumping, and interactions with  collectibles and power-ups.  The player can move left and right within defined lanes and can jump
